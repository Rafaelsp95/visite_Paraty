"Landing Page Turística - Paraty, RJ"

Um projeto web responsivo e moderno desenvolvido para apresentar 
os atrativos culturais e históricos de Paraty. O site foi construído 
com HTML5 semântico e CSS3 avançado, implementando técnicas como 
efeitos overlay, layout flexbox, animações suaves e design responsivo 
para dispositivos móveis.

A página estrutura-se em seções temáticas (Sobre, Pontos Turísticos, 
Galeria) com navegação intuitiva, proporcionando uma experiência visual 
atraente que convida o visitante a conhecer mais sobre este 
Patrimônio Histórico Nacional.

Tecnologias: HTML5, CSS3 (Responsivo, Flexbox, Media Queries)
"

Link do projeto: https://visite-paraty-phi.vercel.app/#sobre
